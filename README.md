# Data Science Portfolio Dashboard

## Overview

This project is an **interactive data analytics dashboard** that allows users to upload datasets and generate insights instantly.
It helps in performing **exploratory data analysis (EDA)**, visualizing trends, and extracting meaningful business insights using an easy-to-use interface.

---

## Objectives

* Perform automated data analysis
* Visualize data using interactive charts
* Provide key insights from datasets
* Build a user-friendly analytics dashboard

---

## Tech Stack

* **Python** (Pandas, NumPy)
* **Visualization**: Matplotlib, Plotly
* **Dashboard**: Streamlit

## How to Run the Project

Follow these steps to run the project locally:

1.  Clone the Repository

```bash
git clone https://github.com/yerrareddy20/data-science-dashboard.git
cd data-science-dashboard
```

2.  Install Dependencies

```bash
pip install -r requirements.txt
```

3.  Run the Dashboard

```bash
streamlit run app/streamlit_app.py
```
